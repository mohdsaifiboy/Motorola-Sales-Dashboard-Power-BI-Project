# Motorola Sales Dashboard-Power BI Project
Power BI sales dashboard analyzing Motorola mobile sales across Indian cities. This project was completed during the # Microsoft Elevate Virtual Internship # using a self-selected dataset

## Project Overview
This project showcases an interactive *Motorola Sales  Dashboard* created using *Power BI*.  
The dashboard provides a complete analysis of sales, quantity, transactions, customer ratings, and payment insights across various Indian cities.


## Dashboard Objectives
- Analyze Motorola product sales across different cities  
- Track monthly and daily sales performance  
- Understand customer behavior and ratings  
- Identify top-performing mobile models  
- Compare payment method preferences  
- Assist business teams in making data-driven decisions
